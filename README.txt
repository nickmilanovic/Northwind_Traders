Database management website built for Northwind Traders with C# on ASP.NET framework using Model-View-Controller template.

Users have the following abilities:
1. Browse through the categories and sort by category or by description in the Categories tab.
2. Filter each product by category, create new products, as well as edit and delete existing in the Products tab.
3. Browse through customer orders and display 5, 10 or 20 orders per page. Pagination applied to the Orders tab.
